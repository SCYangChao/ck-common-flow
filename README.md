# ck-common-flow
  ck-common-flow flowable流程引擎封装 
```
ck-common-flow
├── ck－flow-api  --ＡＰＩ接口服务
├── ck-flow-client  --流程引擎客户端
├── ck-flow-commmon -- 流程引擎公共
├── ck-flow-core -- 流程引擎核心
```
＃后端所用技术
* 系统配置加载与管理：Spring boot 2.0.3.RELEASE
* 代码简化：Lombok 1.16.20
* 接口管理：swagger2　2.8＋
* 工具包：commons-lang3　3.7＋
* JSON序列化：fastjson　1.2.46 
* 数据库：mysql 5.0+
* 数据库持久中间件：MyBatis－plus
* 服务容器：Tomcat　
* 依赖包管理：maven
* 缓存：redis
* 连接池：HikariCP
* 参数校验：hibernate-validator
* 安全认证：oauth2
